# KB_Simulated_Investment
청소년을 위한 교육용 KB 모의 투자

KB 모의 투자는 12가지의 주식 종류로 구성되어 있으며, 매일 뉴스 헤드라인과 해당 종목의 정보를 제공합니다.

회원가입 없이 주식을 매수하고 매도하여 수익률을 비교할 수 있으며, 마지막에는 작년 기준 실제 주식 거래 이용자들의 평균 수익률과 유저의 수익률을 비교합니다.
